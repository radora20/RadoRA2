### 🚀 How to Build APK

1. **Install Prerequisites:**
   ```bash
   # Install Node.js and npm
   # Install Expo CLI: npm install -g expo-cli
   ```

2. **Build APK:**
   ```bash
   cd RemmiaDo-Trading-App
   ./build-apk.sh
   # or manually:
   npm install
   npx expo build:android
   ```

3. **Alternative Testing:**
   ```bash
   npx expo start
   # Use Expo Go app on Android to scan QR code
   ```
